Sample views using scalate templates

  • Jade

    Use ".jade" extention to views filename
    Test here 
                !!!
            -@@ val values:List[Map[String,String]]
            html
              head
                title Sample template with Jade
              body
                table
                  thead
                    tr
                      th Name
                      th Email
                  tbody
                  - for(person <- values)
                    tr
                      td= person("name")
                      td
                        a(href={"mailto:"+person("email")}) #{person("email")}

  • Scaml

    Test here 
                !!!

            %html
              %head
                %title Sample template with Scaml
              %body
                %table
                  %thead
                    %tr
                      %th Name
                      %th Email
                  %tbody
                  - for(person <- values)
                    %tr
                      %td= person("name")
                      %td
                        %a{:href=>{"mailto:"+person("email")}} #{person("email")}
